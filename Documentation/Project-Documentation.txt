-> Cinematic & Television Archive Network Project is a project in which i combine everything i know about Python
-> Cinematic & Television Archive Network Project was developed for the cinematic data lovers
-> Cinematic & Television Archive Network Project will scrape IMDB&TMDB and save the scraped data into a MongoDB and analyse the IMDB cinematic data using streamlit.

One folder called "Cinematic-Harvesting":
    - Contains: 
        - IMDB Folder: All IMDB scrapers for movies and tv-series by genre, one scraper for each.
        - TMDB Folder: One TMDB scraper for movies and tv-series using TMDB API Key.

One folder called Database:
    - Contains: 
        - JSONS Folder: All scraped data it's saved here in a .json format
        - Tools Folder: Python scripts to insert IMDB and TMDB scraped data into MongoDB using .json files

One folder called Documentation:
    - Contains:
        - The project documentation

One folder called Web:
    - Contains the streamlit web app for data visualisation.

3 Separate Python Scripts:
    - FetchAndInsertsIMDB: Starts IMDB movies&tv-series crawlers and inserts the data into MongoDB
    - FetchAndInsertTMDB: Starts TMDB movies&tv-series crawler and inserts the data into MongoDB
    - main.py: Starts FetchAndInsertsIMDB&FetchAndInsertTMDB and the streamlit web app.
